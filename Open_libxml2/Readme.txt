Normally, this should be an empty directory, and you should
use the libxml2 header translations from the following 
download-site:

http://sourceforge.net/projects/libxml2-pas/

If you use cvs to access that site, checkout the module headers.

With getsource on the bash you get a copy of the headers into 
this directory

In the beta1 release I made minor changes to the 
header-translation for delphi 5 compatibility.
